## Penjelasan code
- Program meminta pengguna memasukkan ukuran matriks (baris & kolom).
- Kemudian pengguna memasukkan elemen-elemen kedua matriks.
- Setiap elemen pada posisi [i][j] dijumlahkan:
  hasil[i][j] = matriks1[i][j] + matriks2[i][j];
- Hasil penjumlahan ditampilkan dalam bentuk matriks.
